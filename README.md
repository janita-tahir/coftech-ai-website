# Cofttech.ai - The Future of Innovation through Artificial Intelligence

Modern corporate website designed as a dynamic landing page for an artificial intelligence technology and automation service company. Created using purely semantic HTML5 and personalized CSS3 layout design.

## Features
- **Hero Landing Section:** Interactive dark themed landing hero section with background-blending effects and smooth navigation support.
- **Service Cards:** Grid layout that includes animated hovering service cards on AI development, machine learning, and automation service sections.
- **Product Cards:** Unique dynamic card layout featuring important software including AI Chatbot and analytics platforms.
- **User Contact Form:** Modern contact gateway with validated and clean input fields for user communications.
- **Responsive Design:** Media query optimized to ensure responsiveness and mobile-first layout design at minimum screen sizes of 768px.

##  Tech Used
- **HTML Layout:** Semantic HTML5
- **CSS Style:** CSS3 (Flexbox, Media Queries, Custom Transitions, Background Overlays)
- **Typography:** Segoe UI Fonts & San-Serif Font Scales

##  Directory Structure
├── index.html       # Single page HTML document layout file
├── style.css        # Stylesheet file containing all styles and layouts
└── images/          # Images folder containing asset and portfolio images
